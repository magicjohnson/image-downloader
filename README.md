# image-downloader
Small script to download and save images from list of urls

## Installation

```
$ cd path/to/image-downloader
$ pyvenv ./venv
$ . ./venv/bin/activate
$ pip install -r requirements.txt
```

## Usage: 

```
$ python image_downloader.py urls_example.txt
```

## Run tests:

```
$ pytest tests.py
```
